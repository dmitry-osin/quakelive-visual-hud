How to import and edit your HUD

Right now it's possible to import and edit HUD that was created with Visual HUD application. All HUD data stored in JSON format and saved in in .vhud file.

You can't import code from .menu file simply because there are no single standart for HUD markup/code. There are no strict rules that define name convention, blocks description and so on. It means, that processing/parsing existing HUDs to complicated task and I don't think that anyone should even bother with it.

If you want your old HUD to be editable with VH, you have to rebuild it with VH, download and re-use code, that was generated with application itself. And believe me, it's not that hard, I was able to reproduce standard QL HUDs really quick.

Here is step-by-step instructions on how to import and edit your HUD:

  • Step 1

    • Locate .vhud file provided within downloaded archive, open it with any text editor and copy the code

  • Step 2

    • Click “Load HUD” icon located bottom left

  • Step 3

    • Select “Import custom HUD” from presets drop down
    • Paste the code from .vhud file
    • Click [Load] button
