New features

Extra large editor

Got big screen? Then you are welcome to try brand new, extra large visualHUD editor. To switch between compact editor and large one simply click icon.


New hot keys

When working in extended mode you might need to toggle unnessesary elements on or off. Use the [Tab] key to toggle main toolbar and [F] key to cycle screen modes.


Global status setter

With new visualHUD you can set your armor, health and ammo values from a single place. Move these fancy sliders and see how's all according elements are changing their values.


Improved armor/health bars

Status bars just got better with color ranges support. New you can specify bar colors depending on your health/armor status


Skill-o-meter

Highly demanded feature, small piece of text that shows average skill level of server you are on. New taste with simplified color ranges support
