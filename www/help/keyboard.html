Keyboard shortcuts

Generic Actions

Tab Toggle compact layout view

I Open Import Window

D Open Download Window

Ctrl Shift R Hard reset. All settings will be reverted to their default values

Item Actions Manage selected items with ease

Ctrl A Select all HUD items

Del Delete selected items

Ctrl V Clone selected items

Ctrl Z Undo previous action

Shift G Group/ungroup selected items

Move Items Precisely move items around

← ↑ → ↓ Move selected items by 1 pixel in given direction.

Ctrl → Hold CTRL key to move by 10 pixels

Arrange Items Control items order

Ctrl ↑ Send selected items to back

Ctrl Shift ↑ Send selected items backward

Ctrl ↓ Bring selected items to front

Ctrl Shift ↓ Bring selected items forward
